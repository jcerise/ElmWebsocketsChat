# kemal-chat-elm

A simple example app using Kemal/Crystal and Elm, which exhibits websockets

## Usage

From the root directory, run
```bash
crystal src/kemal-chat-elm.cr
```
Then, in the elm-chat directory,
```bash
elm make Main.elm
```
Followed by
```bash
elm reactor
```

Finally, navigate to localhost:8000
